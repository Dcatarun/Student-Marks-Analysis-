
1. Student Marks Analysis & Description
This project analyzes student marks using Excel, calculates total, percentage, and determines pass/fail status.

2. Objectives
To manage student data efficiently
To calculate total and percentage automatically
To analyze performance using charts and pivot tables

3. Tools Used
Microsoft Excel
Formulas (SUM, AVERAGE, MAX, MIN)
Pivot Table
Charts

4. Project Structure
Explain your sheets:
Sheet 1 (Raw Data): Student marks data
Sheet 2 (Processed Data): Formulas applied
Sheet 3 (Pivot Table): Summary & analysis

5. Features
Automatic calculation of total marks
Percentage calculation
Pass/Fail result
Highest & lowest marks
Dashboard using charts

6. Formulas Used
Give examples:
=SUM(A1:A5)
=AVERAGE(A1:A5)
=MAX(A1:A5)
=IF(percentage>=40,"Pass","Fail")

7. How to Use
Steps:
Download the Excel file
Open in Excel
Enter student marks
Results will be calculated automatically

8. Conclusion
Summarize what you learned:
This project helped in understanding Excel formulas, data analysis, and visualization.
